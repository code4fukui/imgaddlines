# imgaddlines

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A web-based tool to add customizable lines to images.

## Demo
https://code4fukui.github.io/imgaddlines/

## Features
- Add horizontal or vertical lines to an image
- Adjust the spacing, width, and offset of the lines
- Download the modified image

## Usage
1. Drag and drop an image file onto the page
2. Use the sliders to adjust the line spacing, width, and offset
3. Toggle the "Horizontal" checkbox to switch between horizontal and vertical lines
4. Click the "Save" button to download the modified image
