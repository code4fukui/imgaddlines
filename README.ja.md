# imgaddlines

画像にカスタマイズ可能なラインを追加するウェブアプリです。

## デモ
https://code4fukui.github.io/imgaddlines/

## 機能
- 画像にラインを追加できます
- ラインの間隔、太さ、オフセットを調整できます
- 水平/垂直ラインを切り替えられます
- 調整後の画像をダウンロードできます

## 使い方
1. 画像ファイルをドロップします
2. ラインの間隔、太さ、オフセットを調整するためのスライダーを使います 
3. 水平/垂直ラインの切り替えはチェックボックスで行えます
4. 「save」ボタンを押すと調整後の画像がダウンロードできます
