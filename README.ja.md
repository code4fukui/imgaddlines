# imgaddlines

画像にラインを追加するウェブアプリです。

## デモ
https://code4fukui.github.io/imgaddlines/

## 機能
- 画像をドロップしてラインを追加できます
- ラインの間隔、太さ、オフセットを調整できます
- 水平/垂直ラインを切り替えられます
- 調整後の画像をダウンロードできます

## 使い方
画像ファイルをドロップするとラインが追加されます。
ラインの間隔、太さ、オフセットを調整するためのスライダーがあります。
水平/垂直ラインの切り替えはチェックボックスで行えます。
「save」ボタンを押すと調整後の画像をダウンロードできます。
